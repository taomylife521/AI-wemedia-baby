# 1.7 模块与代码文件结构说明

本文档详细介绍了 WeMedia Baby（自媒体宝宝）项目的代码分层架构，并以直观的树状图形式列出了核心代码文件及其具体功能作用。

系统核心代码存放在 `src/` 目录下，整体遵循领域驱动设计（DDD）与整洁架构（Clean Architecture）原则，从内向外分层，确保业务逻辑独立于底层实现与界面展现。

## 代码结构概览树

```text
wemedia-baby/
├── config/                               # [全局配置 / 杂项设置]
│   ├── app_config.json                   # 应用默认参数 (主题外观等)
│   └── feature_flags.py                  # Pro专业版功能开关控制
│
├── src/                                  # [源代码根目录]
│   │
│   ├── domain/                           # [领域层] 核心业务实体与仓储接口 (无依赖)
│   │   ├── models/                       # 业务实体和数据模型定义
│   │   │   ├── account.py                # - 账号模型 (属性: 平台、用户名等)
│   │   │   ├── account_group.py          # - 账号分组模型 (树状/线性结构映射)
│   │   │   ├── publish_task.py           # - 发布任务与记录模型 (发布参数)
│   │   │   └── subscription.py           # - 订阅与会员权益数据模型
│   │   └── repositories/                 # 仓储接口定义 (定义底层数据读写的规范)
│   │       ├── base_repository_async.py  # - 基础异步仓储核心接口
│   │       ├── account_repository_async.py - 账号数据持久化及状态保存接口
│   │       ├── media_file_repository_async.py - 视频媒体文件元信息存取接口
│   │       └── batch_task_repository_async.py - 批量排版及定时任务调度存取接口
│   │
│   ├── services/                         # [服务层] 应用业务流程编排
│   │   ├── account/                      # 账号管理业务中心
│   │   │   ├── account_service.py        # - 账号业务基本增删改查实现
│   │   │   ├── account_manager_async.py  # - 异步框架下的账号池及生命周期调度
│   │   │   ├── account_verifier.py       # - 状态探测与离线/掉线诊断服务
│   │   │   ├── cookie_manager.py         # - 浏览器脱机 Cookie 解析与注入控制
│   │   │   └── login_service.py          # - 执行各平台二维码唤起及验证码拉取流程
│   │   ├── publish/                      # 发布流水线作业核心引擎
│   │   │   ├── publish_service.py        # - 用户意图向前端任务列队的分发中心
│   │   │   ├── publish_executor.py       # - 发布引擎本体 (分配平台插件开始实际执行)
│   │   │   └── pipeline/                 # - 管线化调度: 文件校验 -> 封面上传 -> 标签插入
│   │   ├── file/                         # 本地素材及素材映射管理逻辑
│   │   ├── workspace/                    # 工作区软隔离(用于工作室矩阵号数据拆分)
│   │   ├── subscription/                 # 用户授权检测与权益管理服务
│   │   └── auth/                         # [服务端交互] 登录本系统云端账号的服务
│   │
│   ├── infrastructure/                   # [基础设施层] 数据与技术底层实现
│   │   ├── storage/                      # 本地数据持久化实现方案
│   │   │   ├── tortoise_manager.py       # - 内部高速 ORM (SQLite/TinyDB) 生命周期挂载
│   │   │   ├── file_storage.py           # - 底层文件 IO 控制辅助
│   │   │   ├── backup_manager.py         # - 定期防爆及 SQLite 数据自动快照机制
│   │   │   └── orm_models/               # - Tortoise ORM 各业务的数据建表语句定义
│   │   ├── browser/                      # Playwright 浏览器自动化深层代理及驱动
│   │   │   ├── browser_manager.py        # - 浏览器实例重用管理器及进程守护
│   │   │   ├── browser_factory.py        # - 装配特定指纹/UA/视窗特征的工厂
│   │   │   ├── profile_manager.py        # - 确保各自媒体账号数据隔离存放(独立缓存目录)
│   │   │   ├── fingerprint_checker.py    # - Canvas/WebRTC 绕过与模拟反爬机制
│   │   │   └── human_behavior.py         # - 注入缓动/不规则仿生键鼠曲线操作协助跳过风控
│   │   ├── network/                      # 防火墙穿透及重试策略网关代理控制
│   │   ├── cache/                        # 提供短期状态同步内存缓存 (免查DB加速)
│   │   └── monitoring/                   # 应用层奔溃日志提取及性能打点
│   │
│   ├── ui/                               # [UI展示层] 基于 PyQt / PySide 用户展现
│   │   ├── main_window.py                # 整体窗体 (骨架: 侧边栏/系统托盘菜单/顶部栏)
│   │   ├── pages/                        # 功能重核心视图页
│   │   │   ├── account/                  # - [账号] 列表大视图/搜索/多选功能展现操作
│   │   │   ├── publish/                  # - [队列] 发布流水状态/草稿/记录与参数重发页
│   │   │   │   ├── publish_list_page.py  #   - 列队调度页 (执行/暂停队列)
│   │   │   │   ├── publish_records_page.py - 历史成功失败记录追溯与重试页
│   │   │   │   └── single_publish_page.py- 新建视频发布的填表长页与拖拽配置区
│   │   │   ├── browser_page.py           # - 底层封装轻量级 WebView/WebEngine 辅助展示套壳
│   │   │   ├── file_page.py              # - 云端与本地视频树状分类的交互树
│   │   │   ├── settings_page.py          # - 全局皮肤、默认浏览器特征等配置勾选台
│   │   │   └── workspace_page.py         # - 多工作区可视化创建与登录切换页
│   │   ├── viewmodels/                   # [MVVM桥接层] 双向数据绑定核心
│   │   │   ├── account_viewmodel_async.py# - 接管底层状态异步流同步更新列表树模型
│   │   │   └── workspace_viewmodel.py    # - 控制全局当前域参数实时下分布署
│   │   ├── dialogs/                      # 各场景遮罩弹窗 (警报/信息提示/登录扫码窗/充值窗)
│   │   ├── components/                   # 解耦出的如按钮卡片、分页器等高阶定制组件库
│   │   └── styles/                       # 可变肤 QSS 主题缓存计算规则表
│   │
│   ├── plugins/                          # [插件层 - 动调挂载]
│   │   ├── core/                         # 平台通用行为的基础蓝图接口 (Abstract Classes)
│   │   └── community/                    # 第一方各大主流自媒体短视频适配驱动
│   │       ├── douyin/                   # - [抖音平台] 专属操作脚本库
│   │       │   ├── login_plugin.py       #   - 提供该平台单独截取扫码特征或输入控制
│   │       │   ├── publish_plugin.py     #   - 上传分发流程、平台专属话题@语法爬虫器
│   │       │   └── selectors.py          #   - (热更新) 容纳创作者中心全套 CSS / XPath 映射
│   │       └── kuaishou/                 # - [快手平台] 专属操作脚本库
│   │           ├── login_plugin.py       #   - 适配快手验证流程
│   │           ├── publish_plugin.py     #   - 适配快手上传及定时逻辑
│   │           └── selectors.py          #   - 快手创作者中心定位映射表
│   │
│   └── utils/                            # [杂项工具函数]
│       └── retry.py                      # (例如) 网络挂起后无条件循环重试的万能装饰器
│
└── docs/                                 # 项目文档沉淀中心 (例如本文档所放置的位置)
    └── 01总文档/
        ├── ...
```
