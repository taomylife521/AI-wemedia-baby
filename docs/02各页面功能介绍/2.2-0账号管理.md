# 账号管理功能介绍

## 概述

账号管理是媒小宝的核心功能模块,用于管理所有已绑定的自媒体平台账号。通过账号管理,您可以添加、删除、查看和验证账号,以及管理账号的登录状态和Cookie信息。

## 主要功能

### 1. 账号统计概览

页面顶部展示三个统计卡片:

- **账号总数**: 显示已绑定的账号总数
- **在线账号**: 显示登录状态正常的账号数量
- **离线账号**: 显示需要重新登录的账号数量

### 2. 账号列表管理

#### 2.1 账号列表展示

以表格形式展示所有账号信息,包括:

- **平台**: 账号所属平台(抖音、快手、微信视频号、小红书等)
- **平台昵称**: 账号在平台上的昵称
- **登录状态**: 在线/离线状态,用不同颜色标识
- **最后登录时间**: 最近一次登录的时间
- **操作**: 快速操作按钮(切换、更新、删除等)

#### 2.2 筛选和搜索

- **平台筛选**: 通过下拉菜单筛选特定平台的账号
  - 支持: 全部平台、抖音、快手、微信视频号、小红书
- **关键词搜索**: 在搜索框中输入账号昵称或关键词进行模糊搜索
  - 支持实时搜索(300ms防抖)

### 3. 添加账号

点击"添加账号"按钮,支持以下方式添加账号:

#### 3.1 浏览器登录方式

系统会根据设置中的"浏览器方案"配置,自动选择合适的浏览器类型:

**浏览器方案选择逻辑**:

1. **纯净方案** (Undetected Playwright) - **默认推荐**
   - 使用外置浏览器窗口(独立的Chrome浏览器)
   - 基于Playwright自动化框架,但经过反检测优化
   - 优点: 更接近真实用户行为,不易被平台检测,登录成功率高
   - 缺点: 需要额外的Chrome浏览器,资源占用较大
   - 适用场景: **推荐所有场景使用**,特别是平台风控严格时

2. **混合方案** (Playwright + QWebEngineView)
   - 使用内置浏览器组件(`QWebEngineView`)
   - 浏览器窗口嵌入在应用程序中
   - 优点: 界面统一,操作便捷,资源占用小
   - 缺点: 可能被平台识别为自动化工具,登录成功率较低
   - 适用场景: 资源受限环境,或作为纯净方案的备选

**如何切换浏览器方案**:

1. 进入"设置"页面
2. 找到"浏览器方案"选项
3. 选择"Undetected Playwright (纯净)" (推荐) 或 "Playwright + QWebEngineView (混合)"
4. 重新添加账号时,系统会自动使用新的浏览器方案

> **💡 提示**: 建议使用"Undetected Playwright (纯净)"方案,可以有效避免被平台检测,提高登录成功率。

**添加账号流程**:

1. 点击"添加账号"按钮
2. 选择要添加的平台
3. 系统根据当前浏览器方案自动打开对应的浏览器:
   - **纯净方案** (推荐): 打开独立的Chrome浏览器窗口
   - **混合方案**: 弹出内置浏览器对话框
4. 在浏览器中手动登录账号
5. 登录成功后,系统自动提取Cookie和昵称
6. 点击"保存"按钮完成添加

#### 3.2 支持的平台

- **抖音**: 支持扫码登录和密码登录
- **快手**: 支持扫码登录和密码登录
- **微信视频号**: 支持扫码登录
- **小红书**: 支持扫码登录和密码登录

#### 3.3 自动昵称提取

系统会自动从登录页面提取账号昵称:

- 优先从页面元素提取
- 如果页面提取失败,尝试从Cookie提取
- 如果都失败,使用默认名称"新账号\_{平台}"

### 4. 删除账号

选中一个或多个账号后,点击"删除账号"按钮:

#### 4.1 删除确认

- 弹出确认对话框,显示要删除的账号信息
- 提供"同时删除Cookie和发布记录"复选框(默认勾选)

#### 4.2 删除内容

- **数据库记录**: 删除账号的基本信息
- **Cookie文件**: 删除加密的Cookie备份文件
- **浏览器数据**: 删除浏览器user_data目录(包含Cookie、缓存、Session等)
- **发布记录**: 可选删除该账号的所有发布记录

### 5. 刷新列表

点击"刷新列表"按钮,重新加载账号列表:

- 从数据库获取最新的账号信息
- 更新账号的登录状态
- 刷新统计数据

### 6. 右键菜单操作

在账号列表中右键点击账号,可以执行以下操作:

- **切换账号**: 切换到该账号并打开浏览器
- **更新Cookie**: 手动更新账号的Cookie信息
- **查看详情**: 查看账号的详细信息
- **删除账号**: 删除选中的账号

### 7. 账号验证

系统会自动验证账号的登录状态:

- **Cookie验证**: 检查Cookie文件是否存在且有效
- **HTTP验证**: 通过HTTP请求验证账号是否在线
- **状态更新**: 自动更新账号的登录状态

## 浏览器方案

账号管理支持多种浏览器方案,可在设置中配置:

- **混合方案** (默认): 使用内置浏览器组件
- **纯净方案**: 使用外置浏览器,更接近真实用户行为
- **Playwright方案**: 使用Playwright自动化框架

## Cookie管理

### Cookie存储

- Cookie以加密形式存储在本地
- 存储路径: `AppData/data/{platform}/{username}/backup.encrypted`
- 支持自动备份和恢复

### Cookie清理

删除账号时可选择清理Cookie:

- 删除加密的Cookie备份文件
- 删除浏览器user_data目录
- 确保重新登录时是全新的浏览器环境

## 使用场景

1. **首次使用**: 添加第一个账号,开始使用媒小宝
2. **多账号管理**: 添加多个平台的多个账号,统一管理
3. **账号维护**: 定期检查账号状态,及时处理离线账号
4. **Cookie更新**: 当账号登录过期时,手动更新Cookie
5. **账号清理**: 删除不再使用的账号,释放存储空间

## 注意事项

### 登录安全

- 所有登录操作在本地浏览器中完成,不经过第三方服务器
- Cookie以加密形式存储,保护账号安全
- 建议定期更新Cookie,保持账号活跃

### 删除账号

- 删除账号前请确认是否需要保留发布记录
- 默认勾选"同时删除Cookie和发布记录",避免残留数据
- 删除后无法恢复,请谨慎操作

### 账号状态

- 在线状态不代表账号一定可用,仅表示Cookie有效
- 建议定期手动验证账号状态
- 如果账号频繁离线,可能是平台安全策略导致,建议手动登录

### 昵称提取

- 快手账号需要导航到创作者中心才能提取昵称
- 如果昵称提取失败,可以手动修改账号昵称
- 昵称仅用于显示,不影响账号功能

## 常见问题

### Q: 添加账号后显示"新账号\_{平台}",如何修改?

A: 这是昵称提取失败的默认名称。可以通过以下方式解决:

1. 重新添加账号,确保登录后停留在创作者中心页面
2. 使用"更新Cookie"功能重新提取昵称
3. 手动修改账号昵称(功能开发中)

### Q: 删除账号后重新添加,为什么还是显示原来的账号?

A: 可能是浏览器Cookie未清理干净。解决方法:

1. 删除账号时确保勾选"同时删除Cookie和发布记录"
2. 重启软件后再添加账号
3. 手动清理浏览器user_data目录

### Q: 账号状态显示离线,但实际可以登录?

A: 这是Cookie验证失败导致的。解决方法:

1. 点击"刷新列表"重新验证
2. 使用"更新Cookie"功能更新Cookie
3. 重新添加账号

### Q: 支持哪些平台?

A: 目前支持:

- 抖音
- 快手
- 微信视频号
- 小红书

更多平台正在开发中。
