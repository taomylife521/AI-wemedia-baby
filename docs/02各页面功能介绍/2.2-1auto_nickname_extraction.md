# 自动昵称提取技术文档

> **文档版本**：v1.1
> **更新时间**：2026-03-01

## 概述

自动昵称提取是账号管理模块的核心功能之一,用于在用户登录平台账号后,自动从页面或Cookie中提取账号昵称,避免用户手动输入。本文档按平台分别详细说明提取逻辑,方便后期添加新平台参考和代码查找。

## 技术架构

### 整体流程

```
用户登录 → 提取Cookie → 提取昵称 → 保存账号
                           ↓
                    ┌──────┴──────┐
                    ↓             ↓
              页面元素提取    Cookie提取
                    ↓             ↓
              导航到特定页面   解析Cookie字段
                    ↓             ↓
              执行JS脚本      提取用户信息
                    ↓             ↓
              返回昵称        返回昵称
                    └──────┬──────┘
                           ↓
                      合并结果
```

### 核心组件

| 组件                         | 文件路径                                              | 职责                     |
| ---------------------------- | ----------------------------------------------------- | ------------------------ |
| **PlaywrightBrowserManager** | `src/ui/pages/account/services/playwright_manager.py` | 浏览器管理和昵称提取协调 |
| **DouyinLoginDetector**      | `src/services/account/douyin_login_detector.py`       | 抖音平台检测和提取       |
| **KuaishouLoginDetector**    | `src/services/account/kuaishou_login_detector.py`     | 快手平台检测和提取       |

### 通用提取策略

所有平台都采用**优先级递减**的多层级提取策略:

```
优先级1: 页面元素提取 (最准确)
    ↓ 失败
优先级2: Cookie提取 (备选方案)
    ↓ 失败
优先级3: 默认名称 (兜底方案: "新账号_{platform}")
```

#### 1.1 代码位置

#### 1.2 页面导航逻辑

**关键特性**: 快手平台必须先导航到创作者中心才能提取昵称

```python
# 文件: src/ui/pages/account/services/playwright_manager.py
# 方法: _extract_nickname()

if platform == 'kuaishou':
    current_url = page.url
    creator_url = "https://cp.kuaishou.com/profile"

    # 如果不在创作者中心,先导航过去
    if not current_url.startswith("https://cp.kuaishou.com"):
        logger.info(f"导航到快手创作者中心: {current_url} -> {creator_url}")
        await page.goto(creator_url, wait_until='networkidle', timeout=10000)
        await asyncio.sleep(1)  # 等待页面渲染
```

**为什么需要导航?**

- 快手首页不包含用户昵称信息
- 创作者中心 (`cp.kuaishou.com`) 有明确的用户信息展示区域
- 选择器在创作者中心更稳定可靠

**文本清理与验证**:

```javascript
function cleanAndValidate(text) {
  if (!text) return "";
  text = text.trim();

  // 长度验证 (1-50字符)
  if (text.length >= 1 && text.length <= 50) {
    // 排除明显不是昵称的词
    var invalidKeywords = ["登录", "注册", "创作者服务平台", "首页", "关注"];
    for (var k = 0; k < invalidKeywords.length; k++) {
      if (text === invalidKeywords[k]) return "";
    }
    if (text.startsWith("http")) return "";
    return text;
  }
  return "";
}
```

#### 1.4 Cookie提取

**文件**: `src/services/account/kuaishou_login_detector.py`  
**方法**: `extract_username_from_cookies()`

```python
def extract_username_from_cookies(self, cookies: List[Dict]) -> Optional[str]:
    """从Cookie中提取用户名

    注意: 快手的Cookie中通常不包含明文昵称
    """
    for cookie in cookies:
        if cookie.get('name') == 'userId':
            user_id = cookie.get('value')
            logger.warning("Cookie中未找到明文昵称 (userId不作为昵称使用)")
            return None
    return None
```

> **⚠️ 限制**: 快手平台的Cookie中不包含明文昵称,Cookie提取方案对快手平台效果有限,主要依赖页面元素提取。

#### 2.4 Cookie提取

**文件**: `src/services/account/douyin_login_detector.py`  
**方法**: `extract_user_info()`

```python
def extract_user_info(self, cookies: List[Dict]) -> Dict[str, Any]:
    """从Cookie中提取用户信息"""
    user_info = {
        'username': '',
        'nickname': '',
        'uid': ''
    }

    # 尝试从特定Cookie字段提取
    for cookie in cookies:
        name = cookie.get('name', '')
        value = cookie.get('value', '')

        # 某些Cookie可能包含用户信息
        if 'username' in name.lower():
            user_info['username'] = value
        elif 'nickname' in name.lower():
            user_info['nickname'] = value
        elif name == 'uid_tt' or name == 'uid_tt_ss':
            user_info['uid'] = value

    return user_info
```

**关键Cookie字段**:

```python
KEY_COOKIE_NAMES = [
    'sid_ucp_v1',
    'sessionid_ss',
    'sid_tt',
    'sessionid',
    'passport_csrf_token',
    'passport_csrf_token_default',
    'uid_tt',
    'uid_tt_ss',
    'ttwid'
]
```

## 核心代码实现

### 统一提取入口

**文件**: `src/ui/pages/account/services/playwright_manager.py`  
**方法**: `_extract_nickname()`

```python
async def _extract_nickname(self, context, platform, cookies) -> Optional[str]:
    """提取昵称

    Args:
        context: Playwright浏览器上下文
        platform: 平台标识 (douyin/kuaishou等)
        cookies: Cookie字典或列表

    Returns:
        提取到的昵称,失败返回None
    """
    nickname = None

    try:
        logger.info(f"开始提取昵称: platform={platform}")

        # 1. 页面元素提取
        if context.pages:
            page = context.pages[0]
            logger.info(f"当前页面URL: {page.url}")

            # 快手需要先导航到创作者中心
            if platform == 'kuaishou':
                if not page.url.startswith("https://cp.kuaishou.com"):
                    await page.goto("https://cp.kuaishou.com/profile",
                                  wait_until='networkidle',
                                  timeout=10000)
                    await asyncio.sleep(1)

            # 执行平台特定的提取脚本
            if platform == 'douyin':
                detector = DouyinLoginDetector()
            elif platform == 'kuaishou':
                detector = KuaishouLoginDetector()
            else:
                logger.warning(f"不支持的平台: {platform}")
                return None

            script = detector.generate_nickname_extraction_script()
            result = await page.evaluate(script)

            if result and isinstance(result, dict):
                nickname = result.get('nickname')
                logger.info(f"{platform}昵称提取结果: {result}")

        # 2. Cookie提取(备选)
        if not nickname:
            logger.info("页面提取失败,尝试从Cookie提取...")
            cookie_list = cookies if isinstance(cookies, list) else \
                         [{'name': k, 'value': v} for k, v in cookies.items()]

            if platform == 'douyin':
                detector = DouyinLoginDetector()
                user_info = detector.extract_user_info(cookie_list)
                nickname = user_info.get('username') or user_info.get('nickname')
            elif platform == 'kuaishou':
                detector = KuaishouLoginDetector()
                nickname = detector.extract_username_from_cookies(cookie_list)

    except Exception as e:
        logger.error(f"提取昵称出错: {e}", exc_info=True)

    logger.info(f"昵称提取完成: {nickname}")
    return nickname
```

## 常见问题与解决方案

### 问题1: 快手昵称提取失败

**原因**: 用户登录后停留在快手首页,而不是创作者中心。

**解决方案**: 自动导航到创作者中心 (`https://cp.kuaishou.com/profile`)

**代码位置**: `src/ui/pages/account/services/playwright_manager.py` → `_extract_nickname()`

### 问题2: 提取到无效文本

**原因**: 选择器匹配到了非昵称元素(如"登录"、"注册"等按钮文本)。

**解决方案**: 实现 `cleanAndValidate()` 函数,过滤无效关键词

**代码位置**: 各平台的 `generate_nickname_extraction_script()` 方法中

### 问题3: 页面元素未加载

**原因**: JavaScript脚本执行时,页面DOM尚未完全渲染。

**解决方案**:

- 使用 `wait_until='networkidle'` 等待网络空闲
- 添加 `await asyncio.sleep(1)` 延迟
- 在JavaScript中检查 `document.readyState`

### 问题4: Cookie提取效果不佳

**原因**: 大部分平台的Cookie中不包含明文昵称。

**解决方案**: 优先使用页面元素提取,Cookie提取仅作为备选方案

## 性能优化建议

### 1. 减少页面导航

只在必要时导航到特定页面:

```python
if not current_url.startswith("https://cp.kuaishou.com"):
    await page.goto(creator_url)  # 只有不在创作者中心时才导航
```

### 2. 超时控制

设置合理的超时时间:

```python
await page.goto(url, wait_until='networkidle', timeout=10000)  # 10秒超时
```

### 3. 选择器优化

按优先级排序选择器,找到第一个匹配即停止:

```javascript
for (var i = 0; i < selectors.length; i++) {
  var el = document.querySelector(selectors[i]);
  if (el && cleanAndValidate(el.textContent)) {
    nickname = el.textContent;
    break; // 找到即停止
  }
}
```
