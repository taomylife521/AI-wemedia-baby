# 3.1 抖音插件功能介绍

> **所属分类**：社区插件（开源）
> **源码位置**：`src/plugins/community/douyin/`

本篇文档详细介绍抖音（Douyin）平台的登录与发布自动化插件实现。

---

## 1. 文件结构

```text
src/plugins/community/douyin/
├── __init__.py
├── login_plugin.py          # 登录状态验证与用户信息提取
├── publish_plugin.py        # 自动化内容发布
├── scripts.py               # 注入浏览器的 JavaScript 检测脚本
└── selectors.py             # CSS 选择器常量（昵称元素、关键 Cookie 名等）
```

---

## 2. 登录插件 (`DouyinLoginPlugin`)

抖音登录插件继承 `LoginPluginInterface`，实现了三种验证模式：

### 2.1 浏览器页面在线检测

由 `check_login_status` 方法执行。当浏览器当前页面位于 `douyin.com` 或 `amemv.com` 域名下时：

- 注入 `LOGIN_DETECTION_SCRIPT` 脚本到页面执行。
- 解析脚本返回的 JSON 结果，判断 `loggedIn` 字段是否为 `true`。

### 2.2 用户信息提取

由 `extract_user_info` 方法执行：

- 获取当前页面的所有 Cookies，转换为标准字典格式。
- 从脚本返回体中提取 `username` 字段作为账号昵称。
- 若脚本获取失败，使用 `selectors.py` 中定义的 CSS 选择器作为兜底方案定位页面上的用户名元素。

### 2.3 纯 HTTP 静默验证 (`verify_cookie_http`)

适用于不启动浏览器的后台静默探活场景（如定时刷新账号状态）：

- **请求地址**：`https://creator.douyin.com/aweme/v1/creator/pc/user/info/`
- **请求配置**：自动装配待检测账号的 Cookie 与合法的 User-Agent。
- **有效性判定**：接口返回 `status_code == 0` 且包含 `uid` 或 `user_info` 字段时，确认 Cookie 仍然有效。同时同步更新账号的 `nickname`（或 `unique_id`）。

## 3. 发布插件 (`DouyinPublishPlugin`)

发布插件继承 `PublishPluginInterface`，基于 Playwright 自动化完成抖音创作者平台的内容投递。

### 3.1 表单字段定义

`get_form_schema` 方法返回前端需要渲染的发稿表单字段：

| 字段名        | 类型     | 说明                          |
| ------------- | -------- | ----------------------------- |
| `title`       | 短文本   | 作品标题                      |
| `description` | 多行文本 | 作品描述/文案                 |
| `tags`        | 标签列表 | 发布时的 `#话题` 标签（选填） |

### 3.2 自动发布流程 (`publish`)

1. **登录过期拦截**：导航至 `https://creator.douyin.com/creator-micro/content/upload` 后，若检测到 URL 被重定向至登录页，立即中断并返回"Cookie 已过期"。
2. **素材上传**：定位上传区域（`div[class*="upload"]` 或 `<input type="file">`），通过 `page.expect_file_chooser()` 注入本地素材文件。对于大体积视频，等待最长 **3 分钟** 直到出现"上传成功"提示。
3. **表单填写**：针对抖音的富文本编辑器（`.public-DraftEditor-content`），模拟真实用户操作——点击激活、`Ctrl+A` 全选清空、以 **50ms 按键延迟**逐字打入正文与格式化标签。
4. **提交确认**：点击"发布"按钮后，持续监听页面路由变化（是否跳转至 `**/manage/video**`）或页面弹出"发布成功"提示元素，确认成功后才返回成功状态码。

## 4. 辅助文件

### `scripts.py`

包含 `LOGIN_DETECTION_SCRIPT` 常量——一段注入浏览器执行的 JavaScript 代码，负责检测页面的登录态和提取用户基本信息。

### `selectors.py`

包含 CSS 选择器常量，用于定位抖音创作者中心页面上的关键元素：

- 用户昵称元素选择器
- 关键 Cookie 名称（如 `sessionid`）
