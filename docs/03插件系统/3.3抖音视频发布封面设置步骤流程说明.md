# 3.3 抖音视频发布封面设置步骤流程说明

> **目的**：与流程图一致地说明封面设置步骤的执行逻辑，并给出每步对应的 DOM 点击/操作及步骤完成判定标准。  
> **代码入口**：`src/plugins/community/douyin/steps/step_05_cover_video.py`（CoverVideoStep）  
> **选择器配置**：`src/plugins/community/douyin/selectors.py` → `Selectors.PUBLISH` 中封面相关键。

**方向一弹窗内固定顺序（图1→图2→图3）**：

1. **图1**：打开封面设置弹窗并检测到弹窗后，点击 **「设置横封面」** 按钮（`<span class="semi-button-content">设置横封面</span>`）。
2. **图2**：点击后页面显示设置横封面弹窗状态（横封面预览 4:3、底部红色「完成」按钮等）。
3. **图3**：在弹窗内继续点击 **「完成」** 按钮（`<span class="semi-button-content">完成</span>`），**此时才会触发封面检测程序**；主页面「设置封面」区域出现红框位置的 **「封面效果检测通过」**，即代表封面设置成功。

**两个大方向**（与页面一致）：

- **方向一（本地封面 + 首帧封面）**：点击主页面「竖封面3:4」或「横封面4:3」**上方**的 **「选择封面」** → 进入封面设置弹窗（图1）→ 弹窗内**先点击「设置横封面」**（弹窗切换为图2）→ **再点击「完成」**（触发封面检测）→ 主页面出现「封面效果检测通过」（图3）即成功。
- **方向二（AI 智能推荐封面）**：**不点「选择封面」、不进弹窗**。仅在主页面 **「AI智能推荐封面」** 区域（图 2 红框内）直接点击三个推荐缩略图中的某一个，选为封面 → 进入「封面检测通过」。

**封面检测通过（两个方向相同机制）**：无论方向一还是方向二，步骤完成的判定均为主页面出现 **「封面效果检测通过」** 提示（同一套 DOM / 选择器 `COVER_SUCCESS_INDICATOR`），通过后再下发封面设置成功。

## 2. 步骤完成判定标准（图 3）

**官方成功标准**（图 3 红框所指，两个方向共用）：

- 主页面「设置封面」区域出现 **「封面效果检测通过」** 提示（绿色勾 + 文案），表示封面已设置成功，可进入下一步。**本地/首帧与 AI 使用同一套检测机制。**

**当前代码**：

- 两个方向在各自操作结束后，统一等待主页面出现「封面效果检测通过」提示（`COVER_SUCCESS_INDICATOR`），再下发封面设置成功。对应 DOM 见下文「步骤完成检测」表。

## 3. 各步骤执行逻辑与 DOM/操作

### 3.1 前置条件与分支读取

| 项目     | 说明 |
|----------|------|
| 触发时机 | 步骤 3 上传视频成功（获取到上传成功信号）后执行。 |
| 读取配置 | `metadata["cover_type"]`：`first_frame` / `custom` / `ai`；`metadata["cover_path"]`：本地封面图片路径（当 `cover_type=custom` 时使用）。 |
| 分支逻辑 | `cover_type === "ai"` → 走 **AI 智能封面** 流程（主页面直选）；否则 → 走 **本地/首帧** 流程（点击选择封面 → 弹窗内操作）。 |

| 步骤 | 说明 | DOM / 操作 | 选择器（Selectors.PUBLISH） |
|------|------|------------|-----------------------------|

### 3.4 步骤完成检测（两个方向相同机制）

**重要**：方向一在弹窗内**只有点击「完成」后才会触发封面检测程序**，主页面才会出现「封面效果检测通过」（图3）。方向二在红框内选缩略图后由平台自动触发检测。

无论方向一还是方向二，步骤是否完成均以主页面 **「封面效果检测通过」**（图3 红框）为准，使用同一套 DOM 与选择器。

| 检测项 | 说明 | DOM / 选择器（Selectors.PUBLISH） |
|--------|------|-------------------------------------|
| **封面效果检测通过** | 两个方向共用：主页面出现「封面效果检测通过」提示（绿色勾 + 文案）即视为步骤成功 | **真实 DOM**：`<div class="container-QVu5RH success-container-vgr8T8 coverChecking-fmip_6"><svg class="success-icon-o2IUWQ">...</svg><span>封面效果检测通过</span></div>`。选择器见 `COVER_SUCCESS_INDICATOR`：如 `div.coverChecking-fmip_6`、`div.success-container-vgr8T8:has(span:has-text('封面效果检测通过'))`、`span:has-text('封面效果检测通过')`。 |
| 弹窗已关闭 | 仅方向一：弹窗关闭后再等待上述「封面效果检测通过」出现 | `COVER_MODAL` 选择器在页面上不可见或 count=0。 |

## 5. 流程小结（与页面/图 2 对应）

- **获取到视频上传成功信号** → 由步骤 3 上传成功触发，步骤 5 开始执行。  
- **读取封面设置** → 代码读取 `cover_type` / `cover_path`，分支为「本地/首帧」与「AI」。  
- **方向一（本地/首帧）**：**点击选择封面** → **检测到封面设置弹窗**（图1）→ 弹窗内**先点「设置横封面」**（图2）→ **再点「完成」**（触发封面检测）→ 主页面「封面效果检测通过」（图3）。选择器：`COVER_HORIZONTAL_BTN`、`COVER_CONFIRM_BTN`。  
- **方向二（AI）**：**不点击选择封面、不进弹窗**。仅在主页面图 2 红框区域「AI智能推荐封面」内，用 `COVER_AI_OPTION` 点击三个推荐缩略图之一（通常第一个）。  
- **封面检测通过（两方向相同机制）** → 方向一、方向二完成后，均用同一套逻辑等待主页面出现「封面效果检测通过」（`COVER_SUCCESS_INDICATOR`），再下发封面设置成功。
